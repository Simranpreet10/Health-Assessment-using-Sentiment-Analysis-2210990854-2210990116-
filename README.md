### Project Overview
This project focuses on analyzing text data to assess mental health states using Machine Learning techniques. It utilizes Natural Language Processing (NLP) to categorize sentiments that may indicate specific psychological patterns.

### Repository Structure
To navigate this project, please refer to the following folders:
* **Source Code:** Contains the Jupyter Notebook (`myFile.ipynb`) and the dataset (`Combined Data.csv`).
* **Report:** Contains the Final Project Report and presentation slides.
* **Documents:** Includes the Research Paper and submission screenshots.

### How to Use
1. Navigate to the **Source Code** folder.
2. Open `myFile.ipynb` in Google Colab or Jupyter Lab.
3. Ensure `Combined Data.csv` is in the same directory to run the analysis.
